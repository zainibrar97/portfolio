Projects

What's the Weather?

Weather App using React

A weather app I created using ReactJS. Utilised the Open Weather API to make calls to a restful API and return the current weather data for your city. Using modern Javascript Fetch API, along with React hooks and conditionals, I was able to create a simple interface for the user to immediately get their desired results based on their search for a city.
